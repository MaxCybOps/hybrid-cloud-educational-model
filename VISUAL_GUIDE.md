## THE HYBRID CLOUD ARCHITECTURE IN THE MODEL

```
                  EDUCATIONAL INSTITUTION
                           │
                           │
        ┌──────────────────┴──────────────────┐
        │                                     │
        │    HYBRID CLOUD INFRASTRUCTURE      │
        │                                     │
        │  ┌──────────────┐  ┌─────────────┐ │
        │  │   PRIVATE    │  │   PUBLIC    │ │
        │  │   CLOUD      │  │   CLOUD     │ │
        │  │              │  │             │ │
        │  │ • 4 VMs      │  │ • 2-10 VMs  │ │
        │  │ • Fixed      │  │ • Elastic   │ │
        │  │ • $500/mo    │  │ • $0.096/hr │ │
        │  │              │  │             │ │
        │  │ HIGH         │  │ LOW         │ │
        │  │ SECURITY     │  │ SECURITY    │ │
        │  │ SERVICES:    │  │ SERVICES:   │ │
        │  │              │  │             │ │
        │  │ • Student    │  │ • LMS       │ │
        │  │   Records    │  │ • Library   │ │
        │  │ • Exams      │  │ • Collab    │ │
        │  │ • Admin      │  │             │ │
        │  └──────────────┘  └─────────────┘ │
        │         ▲                 ▲         │
        └─────────┼─────────────────┼─────────┘
                  │                 │
                  │  ROUTING LOGIC  │
                  │  (Predefined)   │
                  │                 │
            ┌─────┴─────────────────┴─────┐
            │                             │
       HIGH SENSITIVITY  →  PRIVATE CLOUD
       LOW SENSITIVITY   →  PUBLIC CLOUD
       MEDIUM SENSITIVITY → DYNAMIC (based on load)
```
